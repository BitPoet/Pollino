Pollino Configuration Settings

The following options are available for Pollino and can be overridden in the $options array passed to renderPoll(). They will also applied on subsequent AJAX calls.

form_action

Default form action

Default action url for the voting form.

Default: 

./

result_sorting

Default result sorting

Default sorting of the answers.

Default: 

sort

show_expiry

Show poll end date if set

HTML Templates

Generic

poll_tpl

Wrapper for poll (form and result)

Default: 

<div class="pollino_poll">
	<div class="pollino_inner">
		<h3>{title}</h3>
		{poll}
	</div>
</div>

expiry_tpl

Wrapper for poll end date

Default: 

<p class="pollino_close">{label}<br>{closedate}</p>

message_tpl

Wrapper for error messages

Default: 

<p class="pollino_error">{message}</p>

Poll Form

form_tpl

Form template

Default: 

<form action="{form_action}" method="get" class="pollino_form">
	<input type="hidden" name="pollino_poll" value="{id}">
	{configField}
	{message}
	{list}
	{button} {loader}
	{closing}
</form>

answer_outertpl

Form choice list wrapper

Default: 

<ul class="pollino_list">{out}</ul>

form_rowtpl

Form row template

Default: 

<li class="pollino_item">
	<label><input type="radio" name="pollino_answer" value="{id}"> {title}</label>
</li>

form_submittpl

Form submit button template

Default: 

<input class="pollino_submit" type="submit" value="{value}">

form_loadertpl

Form loading indicator

Default: 

<span class="pollino_loader pollino_hidden"></span>

Result Display

result_tpl

Result template

Default: 

{message}
{list}
{closed}
{total}

result_outertpl

Result list wrapper

Default: 

<ol class="pollino_list_results">{out}</ol>

result_rowtpl

Result row template

Default: 

<li class="pollino_item pollino_item{key}">
	<span class="pollino_percent_wrapper">
		<span class="pollino_percent_bar stretchRight" style="width:{vote_percent}%;">&nbsp;</span>
	</span>
	{title} ({vote_percent}%, {vote_count})
</li>

total_tpl

Total vote count template

Default: 

<p class="pollino_total">{label} {vote_total}</p>

prevent_voting_type

Prevent Multiple Votings Method

Choose one or the other method

Default: 

use_cookie

cookie_expires

Cookie Lifetime

Lifetime of the cookie in seconds

Default: 

86400

cookie_prefix

Cookie Prefix

Prefix used for the client side cookies

Default: 

pillono_

ip_expires

IP Lifetime

Seconds after the IP restriction will expire. Enter 0 to no expire.

Default: 

86400

use_ua

UserAgent

Additionally use the User Agent string

Default: 

0

user_info

User restriction

Default: 

By using this option. The logged in user will be used to prevent multiple votings. The rest is up to you to handle where and how to render your polls.
